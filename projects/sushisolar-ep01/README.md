## 核心製作策略

### 為什麼選 AIGC

鮨佐樂沒有拍攝素材，也不需要安排重新拍攝。我們的製作方式：

1. **圖生影**：客戶提供現有 IG 食物照片 → Kling AI 轉動態 B-roll
2. **AIGC 新場景**：Midjourney v7 生成傳統拍攝不可能低成本實現的畫面：
   - 日本漁港清晨（食材溯源）
   - 台北大安區夜街（城市感）
   - 師傅背影、手部特寫（職人感，不需真人配合）

### AIGC 工具鏈

```
靜態場景生成  →  Midjourney v7
圖生影動態    →  Kling AI 2.0
攝影機運動    →  Sea Dance 2
後製合成      →  After Effects
音樂          →  Suno AI
旁白（B版）   →  ElevenLabs
```
