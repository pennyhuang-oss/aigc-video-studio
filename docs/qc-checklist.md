# 品質驗收清單

---

## G0 審核清單

- [ ] 四鎖全部確認（規格 / 產品真相 / 品牌資產 / 製作方式）
- [ ] 製作 Brief 第一行已填入成片規格
- [ ] 預算估算完成，核決層級確認
- [ ] 時程表核准
- [ ] AIGC 工具授權狀態確認

---

## G1 審核清單（腳本）

- [ ] 情緒溫度標記完整（每場景有數值）
- [ ] 三段式結構情緒爬升合理（痛點→核心→CTA）
- [ ] 至少一個可量化的具體改變（金額 / 時間 / 人數）
- [ ] 主角動機是個人動機（非「公司要求」）
- [ ] CTA 銜接邏輯已寫明銜接句
- [ ] 數字與真實產品可證的數字對齊
- [ ] 定位一句話與鎖 2 的「這支片賣什麼」一致

---

## G2 審核清單（素材）

- [ ] CHAR 角色臉部一致性通過三要素測試（下巴輪廓 / 眼鏡框 / 鼻樑）
- [ ] 豎屏構圖：主角眼睛在畫面 Y 軸 30% 以內
- [ ] 字幕安全區：不侵入主體視覺區
- [ ] Film grain：σ=8~12，暗部可見顆粒
- [ ] 色溫三段可感受差異（3800K→5000K→5500K 或等效）
- [ ] 所有 AIGC 工具授權確認
- [ ] 第三方 Logo / 商標未出現（或已替換為抽象圖標）
- [ ] Figma Mockup / 未上線功能已加「情境示意」標示

---

## G3 審核清單（精剪）

- [ ] SOT 連續段 ≤12 秒
- [ ] B-roll 上的 SOT ≤1 句
- [ ] UI 展示段之間有人臉反應鏡頭
- [ ] 色溫調色三段可目視區分
- [ ] 音樂四段剪接，段落間淡入淡出 0.5~1 秒
- [ ] GFX 數字動態與 VO 台詞同步（誤差 ≤3 幀）
- [ ] 字幕正確（無錯字、無換行錯誤）
- [ ] 全片浮水印正確顯示
- [ ] 片尾 CTA 只有一個動作 / 一個連結

---

## G4 最終審核清單（品牌一致性）

- [ ] 所有畫面內嵌 URL = 真實網域（含完整後綴）
- [ ] 所有年份對齊上線年度
- [ ] 品牌色碼零容差
  - [ ] 背景色：______（填入色碼）
  - [ ] 強調色：______（填入色碼）
- [ ] Logo 版本正確（深底用白版等）
- [ ] 吉祥物用法符合品牌規範
- [ ] 多平台輸出格式規格全部符合
- [ ] 音量 -14 LUFS 驗證
- [ ] 整體觀感品質得分 ≥ 90
